# Day 1: Foundations — Getting Comfortable with Claude

**Time:** ~40 minutes

## What You'll Learn

1. Navigate the Claude.ai web interface confidently (new chat, conversation history, model selector, settings)
2. Write a clear, complete first message that produces a useful response on the first try
3. Understand what Claude can and cannot do, and where it fits into your marketing/sales workflow

## Today's Materials

| Resource | Description |
|----------|-------------|
| [Lesson](lesson.md) | How-to guide: choosing web vs. desktop, interface tour, connecting GSuite, your first prompt framework |
| [Exercises](exercises.md) | 3 hands-on practice activities using real marketing/sales scenarios |
| [Workflow: First Draft in 5 Minutes](workflow-first-draft.md) | A reusable template you'll use every day after today |

## Suggested Order

1. Read through the lesson (15 min)
2. Complete the exercises in Claude.ai as you go (20 min)
3. Customize the workflow template for your own work (5 min)
