# Day 2: Intermediate Prompting — Getting Exactly What You Need

**Time:** ~43 minutes

## What You'll Learn

1. Use specificity techniques (examples, constraints, negative instructions) to control output quality
2. Load context effectively by pasting in existing documents, data, or style samples
3. Iterate within a conversation to refine output instead of starting over
