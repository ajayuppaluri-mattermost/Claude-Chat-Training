# Day 5: Advanced Features & Team Workflows

**Time:** ~45 minutes

## What You'll Learn

1. Set up and use Projects to maintain persistent context, custom instructions, and shared knowledge
2. Create and use Artifacts for reusable, shareable outputs
3. Understand when to use Cowork and Code beyond standard chat
4. Establish team conventions for consistent, high-quality Claude usage
