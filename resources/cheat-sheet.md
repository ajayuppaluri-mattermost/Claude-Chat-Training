# Claude.ai Cheat Sheet

One-page quick reference. Print this or keep it open while you work.

## The Prompt Formula

**Role + Task + Context + Format**

```
You are a [ROLE].
[TASK — what you need].
Context: [BACKGROUND DETAILS].
Format: [TONE, LENGTH, STRUCTURE, CONSTRAINTS].
```

## Top 10 Refinement Phrases

Use these after Claude's first response to dial in the output:

1. "Make it shorter — under [N] words."
2. "Make the tone more [casual / formal / urgent]."
3. "Rewrite the [first paragraph / CTA / headline] to be more [specific quality]."
4. "Add [a stat / a quote / a deadline] to the [section]."
5. "Remove the [section / paragraph about X]."
6. "Give me 3 alternative versions of the [headline / subject line / CTA]."
7. "Keep everything but change [specific element]."
8. "This is 80% there. Specifically adjust [what needs to change]."
9. "Rewrite this for a [different audience] instead."
10. "Review this for grammar, clarity, and flow. Fix any issues."

## New Chat vs. Continue?

| Situation | Action |
|-----------|--------|
| Structure is right, need detail changes | **Continue** in the same chat |
| Fundamental approach is wrong | **Start new chat** with a better prompt |
| Conversation exceeds 10+ exchanges | **Start new chat** — Claude can lose focus in very long threads |

---

## Projects in 30 Seconds

1. **Create** a Project with a clear name (`Team-Initiative-Quarter`)
2. **Write Custom Instructions** — role, audience, voice, constraints
3. **Upload Knowledge Files** — brand guide, product docs, personas
4. **Every chat in the Project** automatically uses that context

---

## Always Verify

Before using any Claude output:

- [ ] **Facts** — Are specific claims, statistics, and dates accurate?
- [ ] **Tone** — Does it match your brand and the situation?
- [ ] **Completeness** — Did it include everything you asked for?

Claude is a drafting partner, not a fact database. You are the editor-in-chief.
